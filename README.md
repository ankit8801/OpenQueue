# OpenQueue

OpenQueue is a free open-source queue management system for cafes and small restaurants.

It allows customers to join a queue digitally while cafe staff manage tokens in real-time.

Features

• Live queue system
• Token generation
• Real-time updates
• Firebase backend
• Mobile friendly interface

Tech Stack

React  
Firebase Firestore  
TailwindCSS

Use Cases

Small cafes  
Restaurants  
Food courts  
Takeaway counters

Future Plans
